# Demo script (90 seconds)

1. Upload a mixed batch — PDF manual, scanned inspection form, Excel log,
   P&ID drawing.
2. Ask in plain English on the mobile chat: "When was Pump P-201 last
   serviced and what was found?" -> instant answer with cited source.
3. Ask a safety question: "What's the procedure for working near this
   pump?" -> cited procedure appears.
4. Show the compliance dashboard: "3 procedures overdue per OISD-118."
5. Show a lessons-learned alert: "Similar failure pattern detected —
   recommend inspection before next shutdown."
6. Close on the knowledge graph visualisation — click Pump P-201, show its
   entire connected history in one view.
